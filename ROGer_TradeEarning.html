4-1
Account type계좌 유형 선택. Standard(기본) / ECN. 각 칩 호버 시 간단 설명 툴팁 노출
• Standard: 수수료가 스프레드에 포함(별도 커미션 없음), 단순·입문/소액 적합.
• ECN: 원(raw) 스프레드(0에 가까움) + 거래당 커미션, 직접 체결·고빈도/전문가 적합. 기본값 Standard.
4-2
Base currency계좌 기준 통화. 선택 가능: USD(기본, 스테이블코인 기반). 현재 USD 단일 — 추가 통화 지원은 정책 확정 시 드롭다운 노출.
4-3
Leverage레버리지 선택. 선택 가능 옵션:
• 1:50
• 1:100 (기본)
• 1:200
• 1:500
관할/규제에 따라 상한 제한 가능(규제 관할은 1:30 등으로 캡, 미지원 옵션은 비활성/숨김). 선택 드롭다운.
4-4
Trading password거래 터미널(MetaTrader 등) 로그인에 쓰는 거래계좌 비밀번호를 사용자가 직접 설정(확인 입력 포함).
필수 조건:
• 10–15자
• 숫자 1자 이상
• 영문 소문자 1자 이상
• 영문 대문자 1자 이상
• 특수문자 1자 이상 (#[]()@$&*!?|,.^/\+_-)
입력 시 조건별 충족 여부를 ✓/✕ 실시간 표시. ROGer 앱 로그인 비밀번호와는 별개이며 거래 터미널 로그인 시에만 사용.

Account equity계좌 평가액(Equity)과 금일 손익(Today P&L)을 표시. 금일 손익 값만 색상 적용(+gain 초록 / −loss 빨강).
'금일' 기준 시각 = 거래 서버 00:00. 그날 00:00부터 현재까지 거래로 발생한 손익만 집계(범용 FX 산식).
금일 손익 = 실현손익 + 미실현손익 변동 + 스왑 − 수수료
• 실현손익 = 오늘 청산(종료)한 포지션에서 확정된 손익
• 미실현손익 변동 = 보유 중 포지션의 (현재 평가손익 − 오늘 00:00 평가손익)
• 스왑 = 포지션 롤오버(이월)로 발생한 이자
• 수수료 = 오늘 거래에 부과된 수수료(차감 항목)
• 표기 = 계좌 기준통화(base)로 계산 후 USD로 환산
• 금일 손익률(%) = 금일 손익 ÷ 오늘 00:00 시점 Equity × 100
• 미실현 평가 가격 = 종목별 현재가(중간가 mid 기준, 브로커 서버 피드)
• 금액 소수자리 = 표시통화 규칙(USD 2자리)·반올림(round-half-up)
• 갱신 = Equity 카드와 동일 폴링(기본 1초)
