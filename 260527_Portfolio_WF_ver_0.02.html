글로벌 헤더 Portfolio 홈 상단 고정 헤더.
- 상단 행: ROGer 로고(1-1) + 우측 아이콘 그룹(언어 1-2 · 프로필 1-3).
- 하단 행: 페이지 타이틀 "Portfolio".
  • 1-1Logo. ROGer 워드마크. 탭 시 Portfolio 홈 reset
    (이미 본 화면이면 scroll top).
  • 1-2Language. 다국어 제공. 구글 번역 기반으로 현지화 지원 (ROGup 방식 동일).
  • 1-3Profile avatar. 탭 시 front-acc-00 Account Hub(GNB 4)로 이동.
예외 변수
  • 스크롤 시에도 헤더 고정 유지.
2
Demo Mode Toggle (데모 전환 배너) 앱 내 데모 모드와 실거래(Live) 모드를 전환하는 토글 배너.
Demo Mode 활성 시 상단에 앰버 배경 배너가 노출되어 사용자가 모의 데이터 환경임을 인지하게 함.
예외 변수
  • Demo Mode 중 실 주문(Buy/Sell/Withdraw) 버튼 탭 시 → "You cannot place real trades in demo mode." 토스트 노출 후 차단.
  • 앱 재시작 시 Demo 상태 유지(persistent). 명시적 Live 전환 또는 로그아웃 시에만 해제.
  • KYC 미완료 유저: Demo Mode 자동 활성, 토글 비활성화(Live 전환 불가). KYC 완료 후 해제.
⚙️ Demo Mode 설정 위치: Account → Demo (front-dmo-01)
3
Total Value (메인 자산 카드) Cash · Earn · Broker · ROG 평가액을 USD로 합산한 통합 자산 가치.
첫 화면 진입 시 가장 강조되는 핵심 영역.
  • 3-1Amount. 실시간 시세 기반 합산(Cash+투자금+리워드+ROG).
  • 3-2Change. 선택 기간(2-4) 대비 변화량 및 변화율. 탭 시 수익 현황(front-perf-01)로 이동.
  • 3-3Visibility toggle. 금액 마스킹 — 토글 off 시 금액에 상관없이 '•••••'로 노출(실제 값·자릿수 비반영, 고정 5점).
  • 3-4Period. Last 24h(기본) · 7d · 30d · 90d · 1Y · All.(기간에 해당하는 금액 노출)
    선택값은 3-2에 즉시 반영. 총자산 3-1은 항상 현재값.
예외 변수
  • 잔액 $0: "$0.00 USD" 표시 (빈 상태 내 별도 없음)
  • 변화 음수: ↓ 아이콘 + 빨간색 표시
  • 3-3 off 상태: 재진입 시 마스킹 유지 (디바이스 저장)
  • 3-4 기간: 세션 내 유지, 앱 재실행 시 24h 초기화
4
Verification 안내 배너 (신원 인증) 거래 · 입금 · 출금 전 신원 인증(KYC)이 필요함을 알리는 안내 배너.
우측 Verify now › 탭 시 인증 플로우(front-ver-03)로 이동.
예외 변수
  • 인증 완료(Verified) 유저: 배너 숨김(미노출).
  • 인증 진행 중(Pending): "Verification in review" 상태 문구로 대체.
  • Demo Mode: 배너는 노출되나 실제 인증 차단(데모 환경 안내).
5
Quick Actions (바로가기 행) 자금 추가·운용 핵심 단축 2개. (입출금·리워드 전체는 Account 홈에서)
  • 5-1Deposit. 탭 → ROGer_Account.html#front-acc-03 Account 입금(정본).
    Card · Bank · Crypto — 금액·실행은 써드파티.
  • 5-2Earn. 탭 → front-ern-01 Earning 홈(GNB 3).
    Flexible · Fixed-term 예치 진입.
예외 변수
  • 5-2 Earn: Verification 불필요 (잠금 없음)
  • Withdraw · Card · 리워드 수령은 Account 홈(정본)으로 이동 — Portfolio 단축에서 제외
  • Broker 트레이딩 진입은 Quick actions 제외 → Position 행 또는 Trade GNB
