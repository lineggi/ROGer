사용 가이드 · How to read

  • 화면 흐름·뎁스 보기 — 우측 하단 Screen Flow 맵에 마우스를 올리고 상단 GNB 탭을 선택하면 해당 GNB의 화면 흐름(뎁스)이 펼쳐집니다.
  • 프로토타입 이동 — 폰 화면의 버튼·행·아이콘을 클릭하면 실제 연결 화면으로 이동합니다. → 배지가 이동 대상 화면입니다.
  • 넘버 숨기기 — 우측 상단 ◉ 넘버 숨기기 토글로 콜아웃 번호를 켜고 끌 수 있습니다.
  • 콜아웃 1:1 — 폰 UI의 번호 = 우측 디스크립션 번호와 1:1로 매칭됩니다. (점선 X-Y는 하위 항목)
  • 언어 규칙 — 화면(UI) 텍스트는 영문, 기능 명세(디스크립션)는 국문입니다.
